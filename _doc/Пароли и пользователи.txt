Проблемы и их решения:

1. При использовании PHP-7.4 и выше модуль управления тестом выдает предупреждение:
Deprecated: Array and string offset access syntax with curly braces is deprecated in
[путь_к_домашнему_каталогу]\test\class\FastJSON.class.php.

Решение:
а) Использовать PHP-7.2 и ниже.
или:
б) Переписать модуль FastJSON.class.php, заменив {} на [], например:
switch($encode{$pos}) заменить на switch($encode[$pos]).

2. При тестировании на локальной машине модуль управления тестом выдает ошибку:
Warning: mysqli::__construct(): (HY000/2002), ссылаясь на
[путь_к_домашнему_каталогу]\test\sec_mod\security_mod.php on line 5.

Решение:
Вместо адреса типа http://localhost/mytest/test/control.php использовать:
http://127.0.0.1/mytest/test/control.php

3. При использовании виртуальных хостов при попытке авторизоваться выскакивает ошибка.

Решение:
в файле main.php отредактировать ссылку:

$LoginScript=$_SERVER['HTTP_HOST']."/test/login.php";

заменить на:

$LoginScript=$_SERVER['HTTP_HOST']."/mytest/test/login.php";

где "mytest" - имя виртуального теста.
